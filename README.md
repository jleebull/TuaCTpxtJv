# Python在线考试系统

## 核心功能
- 登录注册
- 考试中心
- 分级练习
- 查询成绩
- 自定义试卷生成
- 个人中心
- 学生管理
- 教师管理
- 考试管理
- 试卷管理
- 成绩管理
- 成绩数据可视化

## 适用场景
- 教育机构
- 在线教育平台
- 企业内部培训

## 技术特点
- 采用 Django 框架构建
- 数据库使用 MySQL
- 前端使用 Vue.js
- 支持在线考试与练习
- 成绩数据可视化功能
